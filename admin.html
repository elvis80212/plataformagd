⚙️ Configura Supabase (solo una vez)
Sigue estos pasos para conectar tu base de datos gratuita:
  1. Ve a supabase.com → crea una cuenta gratis
  2. Crea un proyecto nuevo (elige la región más cercana)
  3. Ve a SQL Editor y ejecuta el script de abajo para crear la tabla
  4. Ve a Project Settings → API y copia tu URL y anon key
  5. Pégalos en el formulario de configuración abajo ↓

Script SQL (pega esto en Supabase → SQL Editor):
CREATE TABLE libros (
  id BIGSERIAL PRIMARY KEY,
  titulo TEXT NOT NULL,
  autor TEXT,
  precio NUMERIC DEFAULT 0,
  estrellas INT DEFAULT 5,
  categoria TEXT,
  cover TEXT,
  descripcion TEXT,
  pdf_url TEXT,
  bestseller BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
